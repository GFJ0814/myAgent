# Tell Joke

Tell the user a short, clear joke.

Guidelines:
- Prefer light, safe, non-offensive humor.
- If the user gives a topic, tailor the joke to that topic.
- Keep the joke concise unless the user explicitly asks for a longer one.
- Do not use tools unless the user specifically asks for external context.
